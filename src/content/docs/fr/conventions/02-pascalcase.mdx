**Exemple** : `UserProfile`, `MainController`  

### Pourquoi ?  
- Varie de camelCase mais commence par une majuscule.  
- Permet de distinguer visuellement une **classe** ou **composant** d’une simple fonction.  
- Très utilisé dans **React**, **Angular**, **C#**, **Java**.  

### Exemple pratique

```js
class UserProfile { ... }

function App() {
  return <h1>Hello</h1>;
}
```

Avantages : clarifie l’intention (classe vs fonction).  
Inconvénient : confusion possible si mélangé avec camelCase.
