## camelCase

**Exemple** : `userName`, `getUserAge`  

### Pourquoi ?  
- Lisible car chaque nouveau mot commence par une majuscule.  
- Inspiré du style des langages orientés objet comme Java et C#.  
- Adopté par JavaScript pour les **variables** et **fonctions**.  

### Exemple pratique

```js
let userName = "Alice";
function getUserAge() { return 25; }
```

Avantages : lisible, compact.  
Inconvénient : moins lisible pour les noms très longs.
