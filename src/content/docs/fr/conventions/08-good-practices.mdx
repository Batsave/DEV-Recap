---
title: Bonnes pratiques
description: "Conseils pour appliquer les conventions de nommage efficacement."
sidebar:
  position: 8
---

- Choisir la convention adaptée au **langage et au contexte**.  
- Être **cohérent** dans tout un projet.  
- Ne pas mélanger les conventions au hasard.  
- Préférer des **noms explicites** même longs plutôt que des abréviations cryptiques.  
- Documenter la convention choisie dans un projet collaboratif.  

---

## Exemple mauvais vs bon

### Mauvais :
```css
button.41 { color: red; }
.div2 { margin: 20px; }
```

### Bon :
```css
button.danger { color: red; }
.container { margin: 20px; }
```
