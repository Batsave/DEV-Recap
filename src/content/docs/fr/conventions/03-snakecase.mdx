**Exemple** : `user_name`, `max_value`  

### Pourquoi ?  
- Très lisible grâce aux underscores.  
- Hérité des premiers langages (C, Python).  
- Utilisé dans les **bases de données** (PostgreSQL, MySQL) où la casse est insensible.  

### Exemple pratique

```python
user_name = "Alice"
max_value = 100
```

Avantages : clair, universel.  
Inconvénient : verbeux à taper (`Shift` + `_`).
