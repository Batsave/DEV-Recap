## kebab-case

**Exemple** : `main-container`, `btn-primary`  

### Pourquoi ?  
- Adapté au **CSS et HTML**, car les classes et IDs ne peuvent pas contenir d’espaces.  
- Très lisible dans les fichiers.  
- Utilisé aussi pour les **noms de fichiers**.  

### Exemple pratique

```html
<div class="main-container"></div>
```

```css
.main-container {
  padding: 20px;
}
```

Avantages : lisible, adapté au web.  
Inconvénient : impossible à utiliser en JS (le `-` est interprété comme une soustraction).
