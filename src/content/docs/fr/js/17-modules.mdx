## Exporter un module

```js
// fichier math.js
export function addition(a, b) {
  return a + b;
}

export const PI = 3.14;
```

---

## Importer un module

```js
// fichier main.js
import { addition, PI } from "./math.js";

console.log(addition(2, 3)); // 5
console.log(PI); // 3.14
```

  return `Bonjour ${nom}`;
