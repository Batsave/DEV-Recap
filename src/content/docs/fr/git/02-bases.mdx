## Créer un dépôt

```bash
git init
```

## Cloner un dépôt existant

```bash
git clone https://github.com/utilisateur/projet.git
```

## Vérifier l'état des fichiers

```bash
git status
```

## Ajouter des fichiers à l'index

```bash
git add fichier.txt
git add .
```

## Créer un commit

```bash
git commit -m "Message clair et concis"
```

---

## Bonnes pratiques

- Toujours vérifier `git status` avant un commit.  
- Écrire des **messages de commit explicites**.  
- Commiter souvent, mais avec des changements cohérents.
