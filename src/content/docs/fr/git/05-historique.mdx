## Voir l’historique

```bash
git log
git log --oneline --graph --decorate
```

## Voir les changements

```bash
git diff
git diff --staged
```

## Voir un commit spécifique

```bash
git show abc123
```

---

## Bonnes pratiques

- Utiliser `git log --oneline` pour un historique lisible.  
- Vérifier les changements avec `git diff` avant un commit.
