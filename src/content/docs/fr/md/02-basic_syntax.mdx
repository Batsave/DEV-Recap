## Titres

Markdown gère **6 niveaux de titres** :

```md
# Titre H1
## Titre H2
### Titre H3
#### Titre H4
##### Titre H5
###### Titre H6
```

---

## Mise en forme du texte

```md
**Gras**
*Italique*
~~Barré~~
__Souligné__ (parfois supporté)
`Code inline`
```

### Rendu attendu

**Gras**  
*Italique*  
~~Barré~~  
__Souligné__  
`Code inline`

## Listes

### Liste non ordonnée

```md
- Élément 1
- Élément 2
  - Sous-élément
```

Rendu :  
- Élément 1  
- Élément 2  
  - Sous-élément  

### Liste ordonnée

```md
1. Premier
2. Deuxième
3. Troisième
```

Rendu :  
1. Premier  
2. Deuxième  
3. Troisième
