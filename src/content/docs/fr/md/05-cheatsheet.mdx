| Fonction | Markdown | Rendu | HTML |
|----------|----------|-------|------|
| H1 | `# Titre` | <h1>Titre</h1> | `<h1>Titre</h1>` |
| H2 | `## Titre` | <h2>Titre</h2> | `<h2>Titre</h2>` |
| H3 | `### Titre` | <h3>Titre</h3> | `<h3>Titre</h3>` |
| H4 | `#### Titre` | <h4>Titre</h4> | `<h4>Titre</h4>` |
| H5 | `##### Titre` | <h5>Titre</h5> | `<h5>Titre</h5>` |
| H6 | `###### Titre` | <h6>Titre</h6> | `<h6>Titre</h6>` |
| Gras | `**texte**` | **texte** | `<strong>texte</strong>` |
| Italique | `*texte*` | *texte* | `<em>texte</em>` |
| Barré | `~~texte~~` | ~~texte~~ | `<del>texte</del>` |
| Souligné | `__texte__` | <u>texte</u> | `<u>texte</u>` (support partiel) |
| Code inline | `` `code` `` | `code` | `<code>code</code>` |
| Citation | `> texte` | > texte | `<blockquote><p>texte</p></blockquote>` |
| Liste non ordonnée | `- Élément` | • Élément | `<ul><li>Élément</li></ul>` |
| Liste ordonnée | `1. Élément` | 1. Élément | `<ol><li>Élément</li></ol>` |
| Lien | `[Texte](url)` | [Texte](#) | `<a href="url">Texte</a>` |
| Image | `![alt](img.png)` | (image) | `<img src="img.png" alt="alt" />` |
| Bloc de code | ```js code ``` | Bloc affiché | `<pre><code class='language-js'>…</code></pre>` |
| Table | `| a | b |` | Tableau | `<table>…</table>` |
| Case à cocher | `- [x] fait` | ☑ fait | `<input type="checkbox" checked>` |
