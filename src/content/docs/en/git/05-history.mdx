## View history

```bash
git log
git log --oneline --graph --decorate
```

## View changes

```bash
git diff
git diff --staged
```

## View a specific commit

```bash
git show abc123
```

---

## Best practices

- Use `git log --oneline` for a readable history.  
- Check changes with `git diff` before committing.
