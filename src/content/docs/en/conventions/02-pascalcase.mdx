**Example**: `UserProfile`, `MainController`

### Why?
- Similar to camelCase but starts with an uppercase letter.
- Visually distinguishes a **class** or **component** from a regular function.
- Widely used in **React**, **Angular**, **C#**, and **Java**.

### Practical example

```js
class UserProfile { ... }

function App() {
  return <h1>Hello</h1>;
}
```

Advantages: clarifies intent (class vs. function).
Disadvantage: possible confusion if mixed with camelCase.
