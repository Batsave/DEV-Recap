## Se repérer dans les dossiers

Chaque répertoire est représenté par un **chemin** :  

- `.` → le dossier courant  
- `..` → le dossier parent  
- `~` → le dossier utilisateur  
- `/` → la racine du système  

---

## Commandes principales

| Commande | Description | Exemple |
|----------|-------------|---------|
| `ls`     | Lister le contenu d’un répertoire | `ls` |
| `ls -l`  | Lister avec détails (droits, taille, date) | `ls -l` |
| `ls -a`  | Inclure les fichiers cachés (commencent par .) | `ls -a` |
| `pwd`    | Afficher le chemin complet du dossier courant | `pwd` |
| `cd`     | Se déplacer dans un dossier | `cd Documents/` |
| `cd ..`  | Revenir au dossier parent | `cd ..` |
| `cd ~`   | Aller au dossier utilisateur | `cd ~` |
| `cd /`   | Aller à la racine du système | `cd /` |

---

## Exemple pratique

```bash
cd ~
pwd
ls -la
```

---

## Bonnes pratiques

- Utiliser `ls -la` pour voir tous les fichiers, même cachés.  
- Vérifier son emplacement avec `pwd` avant une commande sensible.  
- Utiliser la touche **Tab** pour l’autocomplétion.
