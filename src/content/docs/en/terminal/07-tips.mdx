## Astuces utiles

- Utiliser les flèches ↑ ↓ pour naviguer dans l’historique.  
- `history` → liste toutes les commandes passées.  
- `!num` → relance une commande depuis l’historique.  
- `alias maj="sudo apt update && sudo apt upgrade"` → créer un raccourci.  
- Utiliser **Tab** pour l’auto-complétion.  
- `clear` ou `Ctrl + L` pour vider l’écran.  

---

## Bonnes pratiques

- Utiliser des **alias** pour gagner du temps.  
- Apprendre quelques **raccourcis clavier** (Ctrl + C, Ctrl + L).  
- Consulter l’historique (`history`) pour rejouer rapidement une commande.
