## Lire des fichiers

| Commande | Description | Exemple |
|----------|-------------|---------|
| `cat`    | Affiche le contenu complet | `cat fichier.txt` |
| `less`   | Lecture page par page | `less fichier.txt` |
| `head`   | Affiche les 10 premières lignes | `head fichier.txt` |
| `tail`   | Affiche les 10 dernières lignes | `tail fichier.txt` |
| `tail -f`| Suit les changements en temps réel | `tail -f log.txt` |

---

## Éditer des fichiers

- `nano fichier.txt` → éditeur simple et intuitif.  
- `vim fichier.txt` → éditeur avancé, puissant mais difficile à maîtriser.  

---

## Bonnes pratiques

- Utiliser `less` plutôt que `cat` pour les fichiers longs.  
- Avec `tail -f`, surveiller les **logs en direct**.  
- Commencer par `nano` avant de se lancer dans `vim`.
