## Main Commands

| Command | Description                                   | Example                            |
|---------|-----------------------------------------------|------------------------------------|
| `ping`  | Test connectivity to an address               | `ping google.com`                  |
| `curl`  | Perform an HTTP request                       | `curl https://example.com`         |
| `wget`  | Download a file                               | `wget https://example.com/file.zip`|
| `ssh`   | Connect to a remote server                    | `ssh user@server.com`              |

---

## Practical Example

```bash
ping -c 4 google.com
curl -I https://openai.com
```

---

## Best Practices

- Use `ping -c <count>` to limit the number of attempts.  
- Check HTTP headers with `curl -I`.  
- Use `ssh-keygen` to set up key-based authentication for passwordless login.
