## Headings

Markdown supports **6 levels of headings**:

```md
# Heading H1
## Heading H2
### Heading H3
#### Heading H4
##### Heading H5
###### Heading H6
```

---

## Text Formatting

```md
**Bold**
*Italic*
~~Strikethrough~~
__Underline__ (sometimes supported)
`Inline code`
```

### Expected rendering

**Bold**  
*Italic*  
~~Strikethrough~~  
__Underline__  
`Inline code`  

---

## Separators

```md
---
***
___
```

Rendering:

---
***
___

---

## Lists

### Unordered list

```md
- Item 1
- Item 2
  - Sub-item
```

Rendering:  
- Item 1  
- Item 2  
  - Sub-item  

### Ordered list

```md
1. First
2. Second
3. Third
```

Rendering:  
1. First  
2. Second  
3. Third
