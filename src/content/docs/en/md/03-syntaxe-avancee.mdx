## Citations

```md
> Ceci est une citation
>> Citation imbriquée
```

Rendu :  
> Ceci est une citation  
>> Citation imbriquée  

---

## Blocs de code

### Sans langage spécifié
````md
```
Texte brut
```
````

### Avec langage (coloration syntaxique)
````md
```js
console.log("Hello World");
```
````

Rendu :

```js
console.log("Hello World");
```

---

## Liens

### Lien simple
```md
[Google](https://google.com)
```

### Lien avec titre (au survol avec la souris)
```md
[Google](https://google.com "Moteur de recherche")
```

### Rendu
[Google](https://google.com)  
[Google](https://google.com "Moteur de recherche")  

---

## Images

```md
![Texte alternatif](image.png)
![Logo](logo.png "Titre au survol")
```

Rendu :  
![Texte alternatif](https://via.placeholder.com/100)
